Dendron lets you create Diagrams using [mermaid](https://mermaid-js.github.io/mermaid/#/)
Create a code block with mermaid support to use mermaid. The following block turns into the diagram below.

<pre>
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
</pre>
