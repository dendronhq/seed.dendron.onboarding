# Summary
- all dates are calculated in milliseoncds from 1970

## Cons
```
var currentTime = new Date();
```

## Property
### getFullYear(): int
- returns 4 digit numbers for years btw 1000 and 9999

### getMonth(): int
- returns month as a zero based value

### getDay(): int
- get day of week
- 0 represents sunday

### getDate(): int
- get date of month
