Dendron uses [Katex](https://katex.org/) to render math.

- Expressions inside `$...$`  will be rendered using inline format.
- Expressions inside `$$...$$` will be rendered using block format.

## Examples

### Inline
```
$f(x) = sin(x)$
```

will render as 

$f(x) = sin(x)$

### Block

```
$$
f(x) = sin(x)
$$
```

will render as

$$
f(x) = sin(x) 
$$
